Title: Notification Service

Participants:
- Administrative Limits DB
- RuleEngine
- Usage Limits
- NotificationQueue
- NotificationService
- Client

Messages:
Administrative Limits DB -> RuleEngine: get configured limits(polling)
Usage Limits -> RuleEngine: get current usage(polling)
RuleEngine -> NotificationQueue: add notification payload when limit is breached
NotificationService -> NotificationQueue: consume
NotificationService -> Client: fire notification

---
Mermaid Diagram Representation:

```mermaid
sequenceDiagram
    participant AL as Administrative Limits DB
    participant RE as RuleEngine
    participant UL as Usage Limits

    UL->>RE: get current usage(polling)

```
